UP2DATE

RITMS UP2DATE is a cloud ready solution for unified software and firmware management. Customers shall use this for software lifecycle management of connected devices where software bundles can be any of the full stack of drivers, firmware and applications.  

Connected devices can use opensource C++/Java/JavaScript/GoLang client libraries as reference implementations of RITMS UP2DATE client API for connected constrained devices. RITMS UP2DATE can be integrated with other products from RITMS product line (such as RITMS IoT Hub, RITMS OS, RITMS Twins, RITMS 61499)

RITMS UP2DATE is based on open and worldwide adopted building blocks, the most important is Eclipse Hawkbit which provides open and flexible Direct Device Integration (DDI) API and Management API.

RITMS UP2DATE extends Eclipse Hawkbit API with zero-cost maintenance device provisioning based on X509 certificates. The Public Key Infrastructure deployed to cloud governs digital certificates to secure end-to-end communications. Devices are automatically provisioned to connect the update service in a secure way.

Use Cases

Connect

Zero-cost registration (commissioning, provisioning) from the cloud when internet connection is on
De-commissioning of connected devices

Public Key infrastructure capable of device and tenant certificate revocation  

Update

Rolling out OS/App Updates over butch of devices which 1) are Initiated and installed from the Cloud (no explicit user confirmation shall be at installation phase, rollout process is fully controlled by cloud operator) or
2) decide when to download/install update (e.g. by HMI confirmation).

Organize

Maintain central software repository with all Firmware/Application releases and tracking information about software status over all fleet of connected devices

Repair

Trace all stages of software installation process on each device including errors and automatic rollouts
