UP2DATE

RITMS UP2DATE - облачное решение для универсального управления ПО подключенных устройств. Используется для централизованного управления жизненным циклом ПО автономных устройств: прошивок и образов ОС, драйверов, конфигураций и приложений.  

Подключенные к облаку устройства используют пакеты и библиотеки клиентов с открытым исходным кодом. Клиенты использующие АПИ RITMS UP2DATE могут дорабатываться под конкретный сценарий использования на  основе эталонных реализаций на языках C++/Java/JavaScript/GoLang включая реализации для устройств с ограниченными ресурсами. RITMS UP2DATE интегрируется с другими продуктами линейки RITMS (RITMS IoT Hub, RITMS OS, RITMS Twins, RITMS 61499)

RITMS UP2DATE построен на промышленно используемых компонентах с открытым исходным кодом, центральный модуль: Eclipse Hawkbit реализует открытые и многофункциональные программные интерфейсы: Direct Device Integration (DDI) API и Device Management (DMI) API .

RITMS UP2DATE дополняет  Eclipse Hawkbit API интерфейсом для автоматического подключения устройств на основе сертификатов безопасности (X509). Ифраструктура публичных ключей, развернутая в облаке управляет доступом к службам по шифрованным каналам и обеспечивает безопасный ввод устроств в эксплуатацию.

Сценарии использования

Подключение в облако

Автоматическая регистрация и ввод в эксплуатацию (настройка) узлов и вычислительных устройств при первом подключении в сеть.

Вывод из эксплуатации управляемый из портала эксплуатирующей организации

Инфраструктура сертификатов обеспечивает полный контроль (права собственности и отзыв сертификатов) для производителей устройств и эксплуатирующих организаций  

Обновление ПО

Массовое развертывание прошивок, образов ОС, приложений и конфигураций ПО на устройствах по команде из облака с опцией автоматической установки или при дополнительном подтвержении оператора устройства.

Управление флотом из единого центра

Репозиторий ПО устройств с поддержкой версионности и гибкими сценариями развертывания на устройствах с учетом аппаратных особенностей, текущего состояния и режима работы устройств.

Удаленное обслуживание

Контроль всех стадий установки обновлений ПО на каждом устройстве. Автоматический откат к предыдущему состоянию в случае сбоя. Доставка ПО устойчивая к аппаратным и сетевым сбоям.

Основные свойства

RITMS UP2DATE распространяется по модели ПО как услуга, готово к использованию при подключении и имеет возможности к расширению и интеграции как программная платформа.

Взаимная сертификация подключений и каналов связи на основе асимметричной криптографии (X509 certificates and mTLS authorization)

Мультитенантная модель авторизации на ролевой основе.

Облачная система резервного копирования и восстановления

Оптимизация ресурсов и стоимости владения облачной услугой

Расширенная аналитика в реальном времени для отдела эксплуатации

Масштабируется до миллионов подключенных устройств

Web UI и программные интерфейсы для автоматизации сценариев

Интерфейсы администрирования через утилиты командной строки

Интеграция с продуктовой линейкой  RITMS для формирования расширенных решений для производителей (собственников) устройств
